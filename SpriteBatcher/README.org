* Usage
** Building
Use nix or cmake. Make sure SDL3 is installed on your system.
#+BEGIN_SRC sh
nix-build
#+END_SRC
** Running
#+BEGIN_SRC sh
cd ./result/bin
./SpriteBatcher
#+END_SRC
